Types and comparison functions

We use optimized implementation for basic types. There are two ways how to use them. First way is to pass cmp_type as comparsion function. Second is to use macro function_type that supplies appropriate function.
qsort(x,xn,sizeof(int),cmp_int) qsort_int(x,xn)
qsort(x,xn,sizeof(char*),cmp_str) qsort_str(x,xn)
qsort(x,xn,sizeof(char*),cmp_casestr) qsort_casestr(x,xn)
